# Corner Blocks

Corner Blocks は、角だけで自分の色をつなげて陣地を広げるブラウザゲームです。ブロック配置ゲームの気持ちよさを、PCでさっと遊べる1人用CPU対戦にしました。

## 遊び方

- あなたは青、CPUは赤です。
- 最初のピースは自分のスタート角に置きます。
- 2手目以降は、自分の色と角で接する場所に置きます。
- 自分の色とは辺で接してはいけません。
- 置けない時はパスできます。
- 全員が置けなくなったら、埋めたマス数が多い方の勝ちです。

## 操作

- 手駒をクリックして選択
- 盤面をクリックして配置
- `回転` でピースを90度回転
- `反転` でピースを裏返し
- `パス` で手番を渡す
- `新しいゲーム` で最初から

キーボードでも `R` で回転、`F` で反転できます。

## 動かし方

`index.html` をブラウザで開くだけで動きます。GitHub Pages を使う場合は、Pages の公開元を `main` ブランチの root に設定してください。
