# PC Codee Lab

PC Codee Lab は、用途と予算からざっくりPC構成を試せる小さな静的Webアプリです。

## できること

- Balanced / Gaming / Creator / Compact のプリセット切り替え
- 予算、CPU、GPU、メモリ、SSD、ケースの調整
- 合計金額、消費電力目安、快適度スコアの表示
- ローカル保存した構成の呼び出し
- 構成メモのクリップボードコピー

## ファイル構成

```text
.
├── index.html
├── styles.css
├── script.js
└── README.md
```

## 動かし方

`index.html` をブラウザで開くだけで動きます。GitHub Pages を使う場合は、Pages の公開元を `main` ブランチの root に設定してください。

## 次に足すと楽しそうなもの

- パーツ名と価格を自分で追加できる編集モード
- 用途別のおすすめ理由をもっと細かく出す診断
- 構成をURLに埋め込んで共有する機能
